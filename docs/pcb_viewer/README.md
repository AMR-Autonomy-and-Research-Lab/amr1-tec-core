# PCB 3D Viewer (tscircuit)

Intento de integración con [@tscircuit/3d-viewer](https://github.com/tscircuit/3d-viewer).  
**Estado:** No compila por dependencias faltantes en el ecosistema tscircuit (miniflex, schematic-trace-solver, etc.).

## Alternativa

Usa **[pcb_viewer.html](../pcb_viewer.html)** para enlaces al playground de tscircuit y recursos PCB.

## Cómo probar tscircuit en el futuro

Si tscircuit corrige sus dependencias:

```bash
npm install
npm run dev
```
