# 🎨 Modelos 3D — AMR TEC Core

Modelos STL para impresión 3D y ensamblaje del robot AMR.

## Contenido

| Ubicación | Archivo | Descripción |
|-----------|---------|-------------|
| `Diseño_CAJA/` | `CajaNuevaAMRF2.STL` | Enclosure/carcasa principal del sistema (v2, casi final) |

Este directorio está reservado para modelos 3D adicionales del proyecto.

## Especificaciones de impresión recomendadas

- **Material:** PLA o PETG
- **Espesor de capa:** 0.2 mm
- **Relleno:** 20–30%
- **Soporte:** Según geometría

## Visualización

Los archivos `.STL` pueden visualizarse con:
- [ViewSTL](https://www.viewstl.com/) (online)
- Blender
- Fusion 360
- PrusaSlicer / Cura
